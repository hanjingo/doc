# 第十一章 支持c++



## 11.1 支持的C++运行库

Android平台带有一个微型的C++运行库支持库，称为系统运行库。该运行库不支持一下特性：

- C++标准库
- 异常支持
- RTTI支持

支持c++的运行库比较

|c++运行库|C++异常支持|c++RTTI支持|C++标准库|
|---|---|---|---|
|系统库|N|N|N|
|GAbi++|N|Y|N|
|STLport|N|Y|Y|
|GNU STL|Y|Y|Y|
