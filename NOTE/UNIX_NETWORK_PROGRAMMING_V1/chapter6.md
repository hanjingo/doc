# 第六章 I/O复用：select和pool函数



## 概述

I/O复用典型使用在下列网络应用场合：

- 当客户处理多个描述符（通常是胡教师输入和网络套接字）时，必须使用I/O复用
- 一个客户同时处理多个套接字是可能的，不过比较少见
- 如果一个服务器即要处理TCP，又要处理UDP，一般就要使用I/O复用
- 如果一个服务器要处理多个服务或者多个协议，一般就要使用I/O复用
