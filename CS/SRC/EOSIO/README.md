# EOSIO项目详解
* eos: eos的整个项目源码；
* eosio.cdt: eos的工具库；
* eosio.contract: eos的官方合约所在，以前在eos中，目前分出来了；
上面3个项目都是fork自:[eosio官方github](https://github.com/EOSIO),添加了自己的注释
