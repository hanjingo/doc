# 大O计数法

[TOC]

## 定义

如果有任何正值常数c和$N_0$时，使得当$N \geq N_0$时，$T(N) \leq cF(N)$，那么我们便可将T(N)的复杂度表示为$O(F(N))^3$



## 常用的函数阶

| 符号        | 名称     |
| ----------- | -------- |
| $O(1)$      | 常数     |
| $O(log n)$  | 对数     |
| $O(n)$      | 线性     |
| $O(nlog n)$ | 现行对数 |
| $O(n^2)$    | 平方     |
| $O(n^3)$    | 立方     |
| $O(c^n)$    | 指数     |
| $O(n!)$     | 阶乘     |
